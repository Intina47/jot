# Security Policy

## Supported Versions

The latest released version of `jot` is the only version that receives
security fixes.

| Version | Supported |
| ------- | --------- |
| Latest release | Yes |
| Older releases | No |

## Reporting a Vulnerability

Do not open public GitHub issues for security vulnerabilities.

Use GitHub's private vulnerability reporting for this repository if it is
available:

- Go to the repository's `Security` tab
- Choose `Report a vulnerability`
- Provide clear reproduction steps, impact, and affected versions

If private reporting is unavailable, contact the maintainers directly through
GitHub and keep the report non-public until a fix is available.

Please include:

- A description of the issue
- Steps to reproduce it
- The affected version or commit
- Any relevant logs, screenshots, or proof of concept

We will aim to acknowledge valid reports promptly, reproduce the issue, and
ship a fix in the next available release.
