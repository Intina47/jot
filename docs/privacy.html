Jot

Privacy Policy

Last updated: April 3, 2026

Overview

Jot is a command-line application. Jot Assistant can connect to third-party services such as Gmail and Google Calendar in order to carry out tasks that the user explicitly requests.

What data Jot accesses

When a user chooses to connect Gmail or Google Calendar, Jot accesses the minimum data needed to provide the requested assistant features, such as reading emails, reading attachment contents, drafting replies, sending replies after confirmation, checking calendar availability, and creating or updating events after confirmation.

Where data is stored

Jot stores configuration files, OAuth tokens, and locally saved output on the user’s device. Jot is designed to operate as a local CLI rather than a hosted inbox service.

How data is used

Data accessed through connected services is used only to fulfill the user’s requested actions inside the application. Jot does not sell user data.

Third-party providers

Jot may send user prompts or selected content to a configured language model provider chosen by the user, such as Ollama. Users are responsible for selecting the model provider they want to use and for understanding that provider’s policies.

Security

Jot attempts to keep access tokens and local configuration files restricted to the current user. No software can guarantee perfect security, and users should protect their devices and credentials accordingly.

Contact

For questions about this policy, open an issue at github.com/Intina47/jot/issues.
