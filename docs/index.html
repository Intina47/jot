jot

Terminal-first tools that stay out of your way.

Jot is a single-binary CLI for writing, file utilities, and a Gmail and Calendar assistant that runs where you already work: the terminal.

View on GitHub Download release

What Jot does

  • Write and shape text from the terminal.
  • Run practical file tasks without leaving the shell.
  • Use jot assistant to read, reply, schedule, and clear across Gmail and Google Calendar.
  • Keep account tokens local to your machine instead of routing email content through a hosted Jot backend.

Jot Assistant

Jot Assistant is designed to make Gmail feel simpler, not heavier. The product is centered on four verbs: read, reply, schedule, and clear. 

jot assistant
jot assistant --onboarding
jot assistant "summarize my unread emails from today"

Distribution

Jot is open source and distributed through GitHub releases. Packaging support exists for Homebrew, npm, and Windows distribution workflows.
